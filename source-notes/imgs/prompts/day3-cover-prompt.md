## 封面图提示词

**符号**：一叠纸 + 压缩弹簧
**点题**：Context 撑爆了？它选择压缩

**提示词**：
```
一幅极简科普漫画封面，温暖平涂风格，浅米色背景，
黑色清晰描边。

画面中央：一叠略显凌乱的对话记录纸（约手掌大小），
纸张之间穿插着一个正在压缩的弹簧符号（⤵），
弹簧把整叠纸压薄了一半，最顶上露出
「compress_context()」字样。

叠纸下方一行粗体小字：「Context 撑爆了？它选择压缩」

除此之外没有任何其他元素。大量留白。
禁止照片写实、3D、暗黑风。扁平矢量。比例 9:5。
```
